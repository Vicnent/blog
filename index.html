Subsurface Analytics

Subsurface Analytics develops and supports deployment of automation, analytics, and machine learning across the business, with a emphasis on statistical enhancement of physics-based modeling, robust regression, time series analysis, real options, and probabilistic graphical models. The goal of this blog is to list examples and use cases of analytics, machine learning, and/or general methods for automation of work tasks, using a variety of tools and data sources.

Contributions from anyone are welcome. Ideally, this is a place for technical discussion of analytics across a diverse audience from Apache business units and technical specialties.

  • Building a Bayesian Network

    David S. Fulford, January 2019

    Bayesian Networks are used to encode the rules, logic, and relationships of multiple variables into a probabilistic graphical model. The visualization of the network describes the variable relationships such as conditional independence and flow of influence. The network allows calculation of likelihood of specific states, or imputing values of missing variables, as primary uses. Here we illustrate with the classic Monty Hall problem.


  • Non-Linear Regression On Production Data

    David S. Fulford, January 2019

    Non-Linear Regression is how we fit arbitary models to data. Some of the challenges in fitting non-linear models involve multiple non-unique fits, local minimums of the cost function, and numerical tolerances. The Arps Hyperbolic model is used to illustrate the concepts on production data.


  • A Visual Primer On Linear Regression

    David S. Fulford, December 2018

    Linear Regression is the fundamental method by which we make any kind of forecast or estimate, including scalar values such as the calculation of a mean or expected value. A greater understanding of the assumptions implicit in linear regression can lead to more robust estimates.


  • The FATHYM_W RAPID Plugin

    David S. Fulford & Derrick W. Turk, November 2018

    The FATHYM_W plugin for the RAPID Engine extends the transient hyperbolic model with a likelihood function which varies the importance of data points over time. This can be used to address concerns like "how can I ensure that RAPID honors late-time behavior over early-time?"


  • Rigline Profiles by Convolution

    David S. Fulford, October 2018

    Generating type well schedules based upon rigline assumptions can be a time-consuming task. This example demonstrates use of the numpy convolution function to generate a rigline scenario from a type well profile and well schedule.
